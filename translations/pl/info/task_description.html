Stephan ma przyjaciela, który jest małym mechbirdem. Ostatnio próbował nauczyć go mówić w podstawowym języku. Dzisiaj ptak wypowiedział swoje pierwsze słowo: "hieeelalaooo". Brzmi to bardzo podobnie do "cześć", ale ma zbyt wiele samogłosek. Stephan poprosił Nikolę o pomoc i pomógł zbadać, jak ptak zmienia słowa. Dzięki informacjom, które odkryli, powinniśmy pomóc im stworzyć moduł tłumaczenia.

Ptak przekształca słowa według dwóch zasad:
  • - po każdej literze spółgłoskowej ptak dodaje losową literę samogłoskową (l ⇒ la lub le);
  • - po każdej literze samogłoski ptak dołącza dwie takie same litery (a ⇒ aaa);
Litery samogłosek == "aeiouy".

Otrzymują Państwo frazę ornitologiczną jako kilka słów, które są oddzielone białymi spacjami (każda para słów jest oddzielona jedną białą spacją). Ptak nie wie, jak interpunkować swoje frazy i wypowiada słowa tylko jako litery. Wszystkie słowa są pisane małymi literami. Proszę przetłumaczyć to zdanie z języka ptaka na coś bardziej zrozumiałego.

Dane wejściowe: Wyrażenie w języku ptaków jako ciąg znaków (str{% if is_js %}ing{% endif %}).

Dane wyjściowe: Tłumaczenie jako ciąg znaków (str{% if is_js %}ing{% endif %}).
