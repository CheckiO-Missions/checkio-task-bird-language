斯蒂芬有一个朋友，他正好是一只小机械鸟。 最近，他正试图教它说基本语言。 今天，这只鸟说出了它的第一个词："hieeelalaooo"。 这听起来很像 "你好"，但元音太多。 斯蒂芬向尼古拉寻求帮助，他帮助研究了这只鸟是如何改变单词的。 利用他们发现的信息，我们应该帮助他们制作一个翻译模块。

这只鸟通过两条规则转换单词：
  • - 在每个辅音字母后，鸟儿会随机添加一个元音字母（l ⇒ la 或 le）；
  • - 在每个元音字母后，鸟儿会附加两个相同的字母（a ⇒ aaa）；
元音字母=="aeiouy"。

你会得到一个鸟类学短语，它是由几个词组成的，这些词之间用白空格隔开 (每对词之间有一个空格)。 这只鸟不知道如何给它的短语加上标点符号，只把单词当作字母说。 所有的词都是小写的。 你应该把这句话从鸟语翻译成更容易理解的语言。

输入：一个字符串的鸟语短语 (str{% if is_js %}ing{% endif %}).

输出：翻译为一个字符串 (str{% if is_js %}ing{% endif %}).
